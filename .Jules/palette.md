## 2024-03-01 - Categorized CLI Help Commands
**Learning:** When a CLI has multiple commands across different domains (Parsing, Validation, Generation), a flat list of commands becomes overwhelming. Organizing commands into functional categories (Processing, Validation, Generation, System) significantly improves Developer Experience (DX) and discoverability.
**Action:** Always group CLI commands into functional categories when the list exceeds 5-7 commands.
