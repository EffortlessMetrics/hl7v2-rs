## 2026-01-27 - CLI Output Readability
**Learning:** Users find raw byte counts (e.g., 8345710592) difficult to parse. Converting to human-readable units (e.g., 7.77 GB) is a high-impact, low-effort micro-UX win.
**Action:** When working on CLI tools, always look for raw numbers (bytes, milliseconds) and format them for human readability using helper functions.
