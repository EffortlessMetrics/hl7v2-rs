This release brings the \hl7v2-rs\ ecosystem to full production readiness.

### Key Highlights
- **Workspace Architecture**: The project has been fully refactored into a scalable workspace of 28 focused microcrates.
- **Server API**: Production-grade HTTP/REST endpoints with constant-time API Key authentication and per-IP rate limiting (\	ower-governor\).
- **Observability**: Integrated Prometheus metrics export (\/metrics\) and detailed structured tracing.
- **Streaming Engine**: Bounded-memory event-based parser available in the CLI via the \--streaming\ flag, capable of securely parsing multi-gigabyte files.
- **Documentation**: A full suite of Architecture Decision Records (ADRs) and comprehensive \README.md\ guides for every crate.
